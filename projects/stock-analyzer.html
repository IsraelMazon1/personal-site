How it works
  1. Ingest and clean historical price data
  2. Compute asset and portfolio returns (log)
  3. Apply weighted portfolio construction
  4. Estimate risk (VaR, CVaR)
  5. Model time-varying risk using rolling windows
  6. Backtest VaR to evaluate calibration
  7. Diagnose distribution behavior (skew, kurtosis, fat tails)
  8. Generate rule-based portfolio insights
  9. Visualize metrics and insights in an interactive dashboard
